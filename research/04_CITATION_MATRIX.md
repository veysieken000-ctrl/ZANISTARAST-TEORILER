# CITATION MATRIX SYSTEM

## PURPOSE

Bu belge, Zanistarast biliminde:

- Teori → yasa → model → uygulama ilişkisini
- Akademik referans sistemini
- Bilimsel doğrulanabilirlik zincirini

standartlaştırmak amacıyla hazırlanmıştır.

Citation Matrix:

Bir teorinin nereden beslendiğini ve nereye uygulandığını gösterir.

---

## 1. CITATION TYPES

### TYPE A — Foundational Citation
Ontolojik veya aksiyomatik temel.

Örnek:
- Theory_01 → Foundational Axioms

---

### TYPE B — Structural Citation
Model kurucu referans.

Örnek:
- Theory_03 → Biological Expansion Model

---

### TYPE C — Transition Citation
Boyut veya evre geçiş referansı.

Örnek:
- Theory_04 → Theory_05 (Reset Transition)

---

### TYPE D — Application Citation
Uygulama ve uygarlık düzeyi referans.

Örnek:
- Theory_07 → Stabilization Engine

---

### TYPE E — Validation Citation
Doğrulama veya test amaçlı referans.

Örnek:
- Unified Equation → Collapse Law Test
