# CITATION STYLE (ZANISTARAST)

## PURPOSE
Provide a consistent citation format across all theory, model, and paper files.

---

## IN-TEXT FORMAT

Use this format inside documents:

- [ZC:AUTHOR_YEAR_TITLE]
- [ZC:SOURCE_ID]

Examples:
- [ZC:Einstein_1916_GeneralRelativity]
- [ZC:Quran_2_255_AyatAlKursi]
- [ZC:Darwin_1859_OriginOfSpecies]

---

## REGISTRY RULE
All citations used in documents must exist in:
citations/01_REFERENCE_REGISTRY.md

---

## MINIMUM FIELDS
Each registry entry must include:
- SOURCE_ID
- TYPE (book/paper/scripture/article/other)
- TITLE
- AUTHOR / ORIGIN
- YEAR (or era if unknown)
- NOTES (why used / scope)

---

## OPTIONAL FIELDS
- edition / translator
- publisher
- URL (optional)
