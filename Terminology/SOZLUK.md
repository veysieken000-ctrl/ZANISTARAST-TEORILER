| Kurdish Term | Turkish | English | Short Meaning |
|---|---|---|---|
| Hebun | Varlık / Ontoloji | Being / Ontology | Varlığın ve yapının alanı |
| Zanabun | Bilgi / Epistemoloji | Knowledge / Epistemology | Bilginin oluşumu ve doğrulanması |
| Mabun | Ekonomi / Akış | Economics / Flow | Kaynakların, değerin ve dolaşımın alanı |
| Rasterast | Yönetim / Kontrol | Governance / Control | Düzenleme, denetim ve yönlendirme alanı |
| Çekirdek | Çekirdek | Core | Sistemin iç yapısı |
| Kontrol | Kontrol | Control | Sistemin bilgiye dayalı yönlendirilmesi |
| Etkileşim | Etkileşim | Interaction | Sistemin çevreyle ilişkisi |
| Sistem | Sistem | System | Çekirdek + kontrol + etkileşim birliği |
| Simülasyon | Simülasyon | Simulation | Modelin çalıştırılarak test edilmesi |
| Bilimsel yapı | Bilimsel yapı | Scientific structure | İddia, model, test ve doğrulama düzeni |
| Bilgi tutarlılığı | Bilgi tutarlılığı | Information consistency | Bilgilerin birbirini bozmaması |
| Çökme oranı | Çökme oranı | Collapse rate | Sistemin kararsızlaşıp dağılma düzeyi |
| Eşitsizlik | Eşitsizlik | Inequality | Kaynak ve dağılım dengesizliği |
| İstikrar | İstikrar | Stability | Sistemin dengede kalabilmesi |
